# Astro Scholar

A minimalist personal academic website for researchers.

Built with [Astro](https://astro.build/) + [Tailwind CSS](https://tailwindcss.com/) + [React](https://react.dev/).

**Features** — Bilingual (EN/ZH) | Dark Mode | Publications | Blog (KaTeX) | Gallery | Interactive Catalogs | Analytics

## Quick Start

```bash
npm install && npm run dev
```

See [CONTRIBUTING.md](./CONTRIBUTING.md) for project architecture, content management, and deployment details.
