## 什么是光谱学？

光谱学是研究物质与电磁辐射相互作用的学科，通过分析辐射的波长或频率来揭示物质的性质。在天文学中，光谱学使我们能够通过分析天体发射或吸收的光来确定其化学组成、温度、密度和运动速度，是天体物理研究中最重要的工具之一。

恒星光度的基本关系由斯特藩-玻尔兹曼定律给出：

$$
L = 4\pi R^2 \sigma T^4
$$

其中 $L$ 是光度，$R$ 是恒星半径，$\sigma$ 是斯特藩-玻尔兹曼常数，$T$ 是有效温度。这个方程构成了恒星天体物理学的基础，将可观测量与恒星的内在性质联系起来。

## 光谱的类型

天体物理学中观测到三种主要类型的光谱：

1. **连续光谱** — 由热的致密源（如恒星内部或加热灯丝）产生的平滑、不间断的色带，覆盖所有可见波长。
2. **发射光谱** — 在特定波长处出现的明亮谱线，当原子中的电子从高能级跃迁到低能级时产生，常见于星云和热气体云中。
3. **吸收光谱** — 叠加在连续光谱上的暗线，当较冷的气体位于光源和观测者之间时，吸收特定波长的光而形成。

## Python 示例代码

以下是使用 Python 加载和可视化光谱的基本示例：

```python
import numpy as np
import matplotlib.pyplot as plt

# 生成合成光谱数据
wavelength = np.linspace(3800, 7200, 1000)  # 单位：埃
flux = np.exp(-0.5 * ((wavelength - 5500) / 800) ** 2)
flux += 0.1 * np.random.randn(len(wavelength))

# 添加吸收线
flux -= 0.3 * np.exp(-0.5 * ((wavelength - 4861) / 5) ** 2)  # H-beta

plt.figure(figsize=(10, 4))
plt.plot(wavelength, flux, 'k-', linewidth=0.5)
plt.xlabel('波长 (Å)')
plt.ylabel('归一化流量')
plt.title('合成恒星光谱')
plt.show()
```

## 展望未来

现代光谱巡天项目如 SDSS、LAMOST 以及即将开展的 4MOST 正在产生数百万条光谱数据，使得跨越银河系及更远范围的恒星族群统计研究成为可能。高分辨率光谱学与机器学习技术的结合正在为我们理解银河系考古学和恒星物理学开辟新的前沿领域。

恒星的径向速度可以通过多普勒效应来测量：$v_r = c \cdot \Delta\lambda / \lambda_0$，这为恒星运动学研究和通过径向速度法探测系外行星提供了关键信息。
